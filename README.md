Схема подключения

Pin 11 → Красный светодиод (+) через резистор 220 Ом Pin 10 → Желтый светодиод (+) через резистор 220 Ом
Pin 9 → Зеленый светодиод (+) через резистор 220 Ом GND → Общий провод (минус всех светодиодов)

Подробное подключение: Красный светодиод: Длинная ножка (анод) → резистор 220 Ом → Pin 11 на Arduino Короткая ножка (катод) → GND на Arduino

Желтый светодиод: Длинная ножка (анод) → резистор 220 Ом → Pin 10 на Arduino Короткая ножка (катод) → GND на Arduino

Зеленый светодиод: Длинная ножка (анод) → резистор 220 Ом → Pin 9 на Arduino Короткая ножка (катод) → GND на Arduino

Настройка Arduino IDE Выбор платы: Откройте Arduino IDE Перейдите в меню Инструменты → Плата → Arduino Uno

Выбор порта: Подключите Arduino к компьютеру USB-кабелем Перейдите в Инструменты → Порт Выберите COM-порт, соответствующий вашей Arduino (обычно COM3, COM4 и т.д.)
